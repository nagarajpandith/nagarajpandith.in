## Local Setup
Clone, Install packages and run the dev server.
```bash
git clone https://github.com/nagarajpandith/portfolio.git
cd portfolio
```

```bash
bun i
bun dev
```

Create a `.env` file similar to `.env.example` and fill the keys.
